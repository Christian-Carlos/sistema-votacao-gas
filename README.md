# Sistema de Votação Online

## Descrição
Sistema desenvolvido com Google Apps Script e Google Sheets para votação online com validação de eleitor, controle de duplicidade e registro de tentativas inválidas.

---

## Funcionalidades
- Validação por CPF e e-mail
- Controle de duplicidade de voto
- Registro de tentativas inválidas
- Integração com Google Sheets
- Interface web personalizada

---

## Tecnologias
- Google Apps Script
- JavaScript
- HTML
- CSS
- Google Sheets

---

## Diferenciais
- Controle antifraude
- Estrutura segura
- Registro de auditoria
- Sistema adaptado para uso institucional
